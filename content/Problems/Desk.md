Nobody can make a good desk. This is a problem for me, as I need a desk, and a good one at that. It needs to be wide and deep, but not thick. These seem like simple requirements of a desk. However, multiple hours of shopping I've done proves you wrong.

This has lead me to the conclusion that making my own desk is the easiest option for me. This will be a design document for that desk, and as I build I will add notes, suggestions, etc, in case you wanna make your own desk, or just understand how I went about mine. 

## Goal Statement
###### Create a desk wide, deep, and thick enough to accommodate my current computer setup while providing ample ample space for schoolwork at the same time. 

##### More specific constraints 
- Width greater than 5 feet
- Width less than 7 feet
- Depth greater than 3 feet 
- Depth less than 4 feet. 
- Thickness able to accommodate mic arm clamp
- Spend <$200

# Special Features
These aren't necessary to the desk experience, but they would be really nice to have...
###### Removable legs
This should be REALLY EASY. Really no matter the material I can put some kind of hardware on the desktop bottom that permits the legs to screw in. 
###### Raised Back
This is a bit harder, particularly if I choose anything other than wood. Wood is just wood. I can cut it however I like, but any other material I'm extremely unfamiliar with cutting/shaping into how I'd like. However, this feature would be so lit. Mostly because if i can do it at all, it means I can ALSO do the next special feature.
###### Speaker Stands
These would be two raised towers on either back corner of the desk for my speakers to go on. Similar roadblocks as the raised back, but if I can figure out one I can certainly do the other. 

### Conclusion 

I will very likely be doing all three of these. Materials willing I don't forsee these being the end of my world. These essentially all build on on another. If I can do the removeable legs on a material less machinable than wood, I can certainly do it again for the raised back and  for the speaker stands. 

# Materials
___
### Wood

| Pros                      | Cons                               |
| ------------------------- | ---------------------------------- |
| Easy to acquire           | Probably kinda expensive           |
| Generally easy to machine | A lot more prep (Sanding/Staining) |
| Grandpa can help LMAO     | Could pretty easily become uneven  |
### Polystyrene or wtv
I cant remember this materials name nor can I look it up on this flight so we go with Polystyrene as a stand-in

| Pros                      | Cons                                              |
| ------------------------- | ------------------------------------------------- |
| Probably decently cheap   | May be hard to machine                            |
| Notably lighter than wood | Less widely avaiiable                             |
| Hard to unlevel           | Unsure how raised back/ speaker stands would work |
| Minimal prep              |                                                   |
### Conclusion 

Probably gonna go with polystyrene. It's cheaper, meaning I'll stay in budget, has very little prep going on, and is hard to fuck up the leveling on. I'm pretty sure amazon has a lot of it in stock for more niche parts, but I do imagine I can find the desktop itself at Home Depot. Special features I want may live and die by what's in stock on Amazon, but I'm willing to risk that.
